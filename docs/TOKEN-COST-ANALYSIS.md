## 五、省钱技巧 💡

### 1. 分阶段使用模型

| 阶段 | 推荐模型 | 原因 |
|------|---------|------|
| 架构设计 | GPT-4/Claude | 需要深度思考 |
| 代码生成 | GPT-4o/Kimi | 速度快成本低 |
| Bug修复 | GPT-4o | 性价比最优 |
| 文档编写 | GPT-3.5 | 足够用了 |

### 2. 控制上下文长度

```
❌ 错误：一个对话里塞满整个项目
✅ 正确：每轮对话清理无关上下文

节省：30-40%
```

### 3. 批量处理

```
❌ 错误：一个个文件生成
✅ 正确：一次性生成相关的一组文件

节省：20-30%
```
