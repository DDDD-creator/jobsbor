# 社交媒体运营方案 - Jobsbor

## 🎯 目标
1. 从社交媒体引流到 jobsbor.vercel.app
2. 建立品牌认知（全球远程工作聚合平台）
3. 获取种子用户（求职者 + 招聘方）

---

## 📱 平台选择

### 第一优先级（必做）
| 平台 | 定位 | 发布频率 | 内容类型 |
|------|------|----------|----------|
| **Twitter/X** | 品牌 + 远程工作资讯 | 每日 2-3 条 | 职位推荐、行业趋势、博客分享 |
| **Telegram 频道** | 社区运营 | 每日 3-5 条 | 最新职位、Web3 资讯、互动 |
| **LinkedIn** | B2B 招聘引流 | 每周 3-4 条 | 行业洞察、招聘趋势 |

### 第二优先级（后续）
| 平台 | 定位 |
|------|------|
| **小红书** | 国内求职者引流（远程工作话题） |
| **Discord** | Web3 社区运营 |
| **YouTube/TikTok** | 短视频：面试技巧、远程工作生活 |

---

## 📝 内容策略

### Twitter/X 内容矩阵
```
40% - 职位推荐（带链接回 Jobsbor）
25% - 行业趋势/博客文章分享
20% - 远程工作技巧/职场建议
15% - 互动话题/投票/ meme
```

**职位推荐：**
```
🔥 新上线：Senior Full Stack Developer @ Web3 Startup

💰 $150k-$200k/year
🌍 Fully Remote
📍 Global

Apply now → https://jobsbor.vercel.app/zh/jobs/[slug]

#RemoteWork #Web3Jobs #Hiring
```
