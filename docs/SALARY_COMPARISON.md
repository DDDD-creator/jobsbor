# 全球薪资对比器 - 功能文档

## 功能概述

全球薪资对比器是一个让用户输入自己的职位、地点和薪资，生成**个性化的薪资对比报告**的工具。核心传播机制是生成精美的分享图片，用户可以将自己的薪资排名分享到社交媒体。

---

## 核心传播机制

### 1. 自我展示心理
- 用户天然有展示自己"优越感"的需求
- "我的薪资超过全球87%的工程师" - 这种表述激发分享欲望
- 百分位排名比绝对数字更具冲击力

### 2. 好奇心理
- "我在全球算什么水平？"
- "我在这个城市是高薪还是低薪？"
- 驱动用户主动使用并分享

### 3. 可视化冲击
- 精美的薪资分布直方图
- 颜色区分的百分位定位
- 专业设计的分享卡片

---

## 功能模块

### 输入模块
```typescript
interface SalaryInput {
  jobTitle: string;      // 职位选择（支持模糊搜索）
  location: string;      // 城市/国家选择
  salary: number;        // 当前年薪（可选）
  experience: string;    // 工作年限
  currency: string;      // 货币类型
}
```

### 数据处理
- 从多个数据源聚合薪资数据
- 根据用户选择进行过滤和计算
- 计算全球/全国/本地百分位排名

### 可视化输出
1. **薪资分布直方图**
   - X轴：薪资区间
   - Y轴：人数分布
   - 用户位置标记（高亮显示）

2. **百分位排名卡片**
   - 大字体显示排名
   - 颜色编码（绿色=优秀，红色=需提升）

3. **分享图片**
   - Jobsbor Logo
   - 用户排名
   - 二维码
   - 简短URL

## 技术实现

### 前端组件
```
/tools/salary-comparison/
├── page.tsx                    # 主页面
├── components/
│   ├── SalaryInputForm.tsx     # 输入表单
│   ├── SalaryChart.tsx         # 薪资分布图
│   ├── PercentileDisplay.tsx   # 百分位展示
│   └── ShareCard.tsx           # 分享卡片
└── hooks/
    └── useSalaryData.ts        # 数据获取
```

### 图片生成
- 使用 `html-to-image` 库
- 预渲染分享卡片DOM
- 转换为PNG下载

### 数据API
```
GET /api/salary-data?job={jobId}&location={locationId}
Response: {
  distribution: SalaryBin[],
  percentiles: {
    p10: number,
    p25: number,
    p50: number,
    p75: number,
    p90: number
  },
  metadata: {
    sampleSize: number,
    lastUpdated: string
  }
}
```
