# 网站内容优化任务文档

## 目标
优化Jobsbor网站内容，提升用户体验和SEO效果

## 内容优化方向

### 方向1: 岗位详情优化
**负责人**: 内容优化智能体1

#### 任务清单
1. **优化岗位描述**
   - 每个岗位增加"岗位亮点"bullet points
   - 补充公司背景介绍
   - 添加职业发展路径说明

2. **丰富职位要求**
   - 区分"必须技能"和"加分项"
   - 添加软技能要求
   - 补充团队文化描述

3. **增加薪资透明度**
   - 添加薪资构成说明（Base+Bonus+期权）
   - 标注薪资竞争力（高于市场平均xx%）
   - 补充福利详情

4. **SEO关键词优化**
   - 标题包含目标关键词
   - 描述中自然融入长尾词
   - 标签精准化

### 方向2: 页面内容填充
**负责人**: 内容优化智能体2

#### 任务清单
1. **首页内容增强**
   - Hero区域添加统计数据（已入驻公司数、月活职位数）
   - 添加"热门行业"快速入口
   - 增加用户 testimonial/评价

2. **关于我们页面**
   - 平台使命和愿景
   - 团队介绍
   - 合作伙伴展示

3. **行业报告/博客**
   - Web3招聘市场趋势报告
   - 大厂薪资对比分析
   - 面试技巧系列文章

4. **帮助中心**
   - 求职指南
   - 简历优化建议
   - 面试准备清单

## 内容规范

### 写作风格
- 专业但不生硬
- 鼓励性语言
- 数据支撑观点
- 段落简洁（不超过3行）

### SEO要求
- 标题字数: 30-60字符
- 描述字数: 120-160字符
- 关键词密度: 2-3%
- 使用H2/H3层级清晰

### CTA植入
每个页面必须包含:
- 主CTA: 浏览职位按钮
- 副CTA: 加入TG频道 https://t.me/Web3Kairo

## 输出格式
- TypeScript数据文件 (.ts)
- Markdown文章 (.md)
- 组件代码片段 (.tsx)

## 优先级
P0: 岗位详情优化（直接影响转化）
P1: 首页内容增强（影响第一印象）
P2: 博客/报告（长期SEO价值）
P3: 帮助中心（用户体验）
