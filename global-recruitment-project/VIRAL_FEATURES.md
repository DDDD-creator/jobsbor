## 功能1：全球薪资对比器 💰

### 核心概念
让用户输入自己的职位和地点，生成**个性化的薪资对比报告**，显示其薪资在全球/本地的百分位排名。

### 为什么能传播
1. **自我展示心理** - 人们喜欢展示自己的"优越感"
2. **好奇心理** - "我在全球算什么水平？"
3. **可视化冲击** - 精美的图表天然适合分享

### 功能特性

#### 输入
- 职位选择（支持模糊搜索）
- 城市/国家选择
- 当前薪资（可选，用于精准定位）
- 工作年限

#### 输出
- 📊 薪资分布直方图（带用户位置标记）
- 🏆 全球/全国/本地百分位排名
- 📈 同职级薪资区间
- 💡 建议：跳槽溢价、技能溢价分析

#### 分享机制
- **一键生成分享图**：精美的PNG卡片
- **图片包含**：
  - Jobsbor Logo + 二维码
  - 用户排名（"你的薪资超过全球87%的工程师"）
  - 简短URL：jobsbor.com/s/xxxxx
- **社交文案模板**：
  - "我在全球工程师中排名前13%，你呢？"
  - "原来我在上海算是高薪了..."

### 技术实现
```typescript
// 核心组件
- SalaryComparisonInput    // 输入表单
- SalaryDistributionChart  // 薪资分布图
- ShareCardGenerator       // 分享卡片生成
- LocationSelector         // 地理位置选择
```

### 数据来源
- Levels.fyi API（科技公司薪资）
- Glassdoor 数据
- Payscale 全球数据库
- 用户自报数据（匿名聚合）

## 功能2：面试题库 📚

### 核心概念
按公司、职位、难度分级的面试题库，支持用户贡献题目，形成UGC生态。

### 为什么能传播
1. **备考刚需** - 面试前的必备工具
2. **打卡分享** - "今天刷了5道Google题"
3. **挑战机制** - 好友PK答题

### 功能特性

#### 分类体系
```
📁 按公司
  ├─ FAANG
  │   ├─ Google
  │   ├─ Meta
  │   ├─ Amazon
  │   └─ ...
  ├─ 国内大厂
  │   ├─ 字节
  │   ├─ 阿里
  │   └─ ...
  └─ 外企
      └─ ...

📁 按职位
  ├─ 前端
  ├─ 后端
  ├─ 算法
  ├─ 产品
  └─ 设计

📁 按难度
  ├─ 🟢 初级（0-2年）
  ├─ 🟡 中级（3-5年）
  └─ 🔴 高级（5年+）
```

#### 题目详情
- 题目描述
- 难度标签
- 出现频率（基于面经统计）
- 参考答案
- 讨论区
- 相关题目推荐

#### 用户功能
- 📝 收藏题目
- ✅ 标记已掌握
- 📊 学习进度统计
- 🏆 连续打卡天数
- 💬 评论和讨论

#### 分享机制
- **刷题打卡**："连续刷题7天！"
- **挑战好友**：生成挑战链接
- **题目分享**：分享特定难题求助
- **成就徽章**："Google题霸"、"算法王者"

### 技术实现
```typescript

- QuestionDetail         // 题目详情
- StudyProgress          // 学习进度
```

### 内容策略
- 初期：爬取公开面经，整理经典题
- 中期：引导用户贡献，积分激励
- 长期：社区自运转，高质量UGC

## 功能3：内推码系统 🔗

### 核心概念
内推码交换平台，用户提交内推码，查看成功率，内推人排行榜，形成网络效应。

### 为什么能传播
1. **互惠互利** - 你帮我内推，我帮你内推
2. **排行榜竞赛** - 争夺"内推王"称号
3. **返利机制** - 成功内推获得奖励

### 功能特性

#### 提交内推码
- 选择公司
- 填写内推码
- 设置有效期
- 备注（部门、城市、限制条件）

#### 内推码展示
- 公司Logo墙
- 剩余次数显示
- 成功率统计
- 最后更新时间

#### 排行榜系统
```
🏆 本周内推王
🥇 张三 - 成功内推12人
🥈 李四 - 成功内推8人
🥉 王五 - 成功内推5人

🏅 总榜TOP10
💎 钻石内推官（50+成功）
🥇 金牌内推官（20+成功）
🥈 银牌内推官（10+成功）
🥉 铜牌内推官（5+成功）
```

#### 成功追踪
- 用户标记"内推成功"
- 内推人获得积分
- 被内推人入职后确认

#### 分享机制
- **我的内推码**：分享到社交平台
- **排行榜炫耀**："我是本周内推王！"
- **互助邀请**："求字节内推，可换阿里内推"

### 技术实现
```typescript
// 核心组件
- ReferralSubmitForm     // 提交内推码
- ReferralCodeCard       // 内推码卡片
- ReferralLeaderboard    // 排行榜
- SuccessTracker         // 成功追踪
- CompanyReferralWall    // 公司内推墙
```

### 防作弊机制
- 验证码使用限制
- 成功率异常检测
- 用户举报机制
- 内推人身份验证

## 功能4：远程工作筛选器 🌍

### 核心概念
全球远程职位聚合，时区匹配，薪资货币转换，签证/税务信息一站式查询。

### 为什么能传播
1. **数字游民热点** - 远程工作是趋势
2. **实用工具** - 解决时区、货币、签证痛点
3. **生活方式向往** - "在巴厘岛写代码"

### 功能特性

#### 职位筛选
- 🌍 时区匹配（与我的时区差 ≤ X小时）
- 💰 薪资范围（转换为本币）
- 🏢 公司类型（Startup/大厂/非营利）
- 🌐 语言要求
- 📄 签证支持

#### 智能匹配
- 输入你的时区 → 显示友好时区职位
- 输入期望薪资 → 自动换算各国货币
- 输入国籍 → 显示可签证支持的公司

#### 实用工具
- **时区重叠计算器**：显示与团队的共同工作时间
- **薪资换算器**：实时汇率，购买力平价
- **签证指南**：各国数字游民签证信息
- **税务计算器**：粗略估算税务负担

#### 分享机制
- **匹配结果**："找到15个适合我的远程职位"
- **时区可视化**：分享时区重叠图
- **数字游民攻略**：签证/税务经验分享

### 技术实现
```typescript
// 核心组件
- RemoteJobFilter        // 职位筛选器
- TimezoneMatcher        // 时区匹配
- SalaryConverter        // 薪资换算
- VisaGuide              // 签证指南
- TaxEstimator           // 税务估算
```

### 数据来源
- RemoteOK API
- We Work Remotely
- LinkedIn 远程职位
- 各公司招聘页面

## 功能5：技能评估测试 🧠

### 核心概念
技术栈能力测试，生成可视化能力报告，推荐匹配职位，支持好友PK。

### 为什么能传播
1. **炫耀心理** - 展示自己的能力
2. **求职刚需** - 了解自己的能力定位
3. **游戏化** - 闯关、排名、PK

### 功能特性

#### 测试类型
```
🎯 技术栈测试
  ├─ 前端技术栈（React/Vue/Angular/CSS）
  ├─ 后端技术栈（Java/Go/Python/Node）
  ├─ 算法能力
  ├─ 系统设计
  ├─ 产品思维
  └─ 数据科学

⏱️ 测试形式
  ├─ 单选题（基础知识）
  ├─ 多选题（综合应用）
  ├─ 代码题（在线编程）
  └─ 场景题（系统设计）
```

#### 能力报告
- 📊 雷达图：各项能力评分
- 📈 百分位排名：超过X%的测试者
- 💪 优势技能：你的强项
- 📚 提升建议：推荐学习资源
- 🎯 匹配职位：根据能力推荐

#### 游戏化元素
- 🏆 等级系统：青铜→白银→黄金→铂金→钻石→王者
- 🎖️ 徽章系统："React专家"、"算法王者"
- ⚔️ 好友PK：发起挑战，一决高下
- 📊 排行榜：全球/好友排名

#### 分享机制
- **能力证书**：生成精美证书图片
- **等级炫耀**："我达到钻石段位了！"
- **PK挑战**："来挑战我，看谁能赢"
- **匹配职位**："我的能力匹配这些高薪职位"

### 技术实现
```typescript
// 核心组件
- SkillTestQuiz          // 测试界面
- AbilityRadarChart      // 能力雷达图
- SkillCertificate       // 技能证书
- FriendPK               // 好友PK
- Leaderboard            // 排行榜
- JobMatcher             // 职位匹配
```

### 题库建设
- 初期：整理经典面试题
- 中期：用户贡献题目
- 长期：AI生成个性化题目
