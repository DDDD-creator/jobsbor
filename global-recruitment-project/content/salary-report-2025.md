## 📊 核心发现

### 金融 vs Web3 薪资对比

| 行业 | 平均年薪 | 中位数 | 最高薪资 | 奖金/代币 |
|------|---------|--------|---------|----------|
| **传统金融** | $185,000 | $160,000 | $500,000+ | 20-100% |
| **Web3/加密** | $220,000 | $195,000 | $400,000+ | 50-200% |
| **Quant/高频** | $350,000 | $300,000 | $1,000,000+ | 100-300% |

**关键洞察**：Web3行业基础薪资比传统金融高19%，但量化金融的总包（含奖金）仍然是全行业最高。

## 🎯 求职建议

### 想进顶级金融公司？

1. **学历**：Target School优先（常春藤、LSE、牛剑）
2. **技能**：Python + 统计 + 金融建模
3. **路径**：实习 → 全职 → 跳槽涨薪
4. **时机**：秋季招聘（9-11月）是关键

### 想进Web3公司？

1. **技能**：Solidity + React + 智能合约安全
2. **作品集**：GitHub上有DeFi项目贡献
3. **社区**：参与Discord治理，建立声誉
4. **时机**：全年招聘，熊市是最佳入场时机

### 薪资谈判技巧

| 策略 | 效果 |
|------|------|
| 多offer竞争 | 可涨薪20-30% |
| 强调代币激励 | Web3公司总包提升30%+ |
| 远程工作谈判 | 可抵消地域薪资差异 |
| 签约奖金 | 一次性$10k-$50k |
