## 清洗规则

### 去重规则
1. **完全去重**: 相同公司 + 相同职位名称 = 重复
2. **相似去重**: 职位相似度 > 80% = 重复
3. **公司去重**: 相同公司只保留一条记录

### 清洗规则
1. **薪资格式化**: 统一转换为 "xxk-xxk" 格式
   - BOSS直聘/拉勾网/智联招聘: 月薪（元）→ k
   - 猎聘: 已是k单位，直接使用
   - Web3(USD年薪): 转换为人民币月薪k
2. **地点标准化**: 统一为城市名（去除区县）
3. **标签规范化**: 保留原始标签，统一格式
4. **日期格式化**: 统一为 YYYY-MM-DD 格式
