## 🎯 内链策略说明

### 1. 金字塔型内链结构
```
                    首页
                     │
        ┌────────────┼────────────┐
        │            │            │
    职位列表    公司列表      博客列表
        │            │
    职位详情    公司详情
```

### 2. 相关内容推荐策略
- **横向关联**: 同行业职位/公司
- **纵向关联**: 公司↔职位双向链接
- **上下文关联**: 面包屑导航显示层级

### 3. 权重传递路径
```
首页 (最高权重)
  │
  ├─→ 职位列表 ──→ 职位详情 ──→ 公司详情
  │
  ├─→ 公司列表 ──→ 公司详情 ──→ 职位详情
  │
  └─→ 博客列表 ──→ 文章详情
```
